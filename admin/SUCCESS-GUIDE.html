📋 使用指南

🚀 第1步：登录管理后台

  1. 点击上方"立即进入管理后台"按钮
  2. 输入您的管理员用户名：xwteam
  3. 输入您的管理员密码
  4. 点击"登录管理后台"

📝 第2步：编辑网站内容

  1. 在左侧菜单选择要编辑的页面配置
  2. 使用可视化表单编辑内容
  3. 支持展开/收缩功能，方便管理大量内容
  4. 点击"保存配置"即时生效

🛍️ 第3步：管理产品信息

  1. 选择"产品配置"
  2. 添加新产品或编辑现有产品
  3. 产品ID会自动生成时间戳
  4. 分类名称和值会自动同步

🔄 第4步：同步到仓库

  1. 编辑完成后，可选择"同步到Gitee"
  2. 所有修改会自动备份到Gitee仓库
  3. 提供版本控制和数据安全保障
